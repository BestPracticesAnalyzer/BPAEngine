The Instance Element

This element represents a single instance of an object query.  An object query might match multiple items, there will be one Instance element per matching item.  Instance elements are only generated on output, and they have almost the same schema as Object elements, except the properties that are not changed on output will not be included in each Instance element. 

Example Configuration Fragment

In an output data file:

<Object
   Type="WMI"
   Key1="%SRVNAME%"
   Key3="Win32_Service"
   ProcessTime="973.6472"
>
     <Instance
         Type="WMI"
         Key1="EXASIA02.disjtop.com"
         Key3="Win32_Service"
         Name="root\cimv2\Win32_Service.Name=1-vmsrvc"
     >
     ...

Parent Elements

Element Description
Object Instance elements are output only and are always nested under the Object that created them.

Child Elements

Element Description
Setting This element contains data about a particular retrieved property.

Attributes

Attribute Description
Key1 - Key5 These are the same as the object, except substitutions are filled in.
Name This provides information on the retrieved name of the instance.
Type This is the type of the object that created this instance.
