Best Practices Analyzer Core Library
The GetUserNameEx Object Processor

This retrieves extended information for the current logged on user name.

Example Configuration Fragment

The ObjectProcessor definition:

<ObjectProcessor
   Name="GetUserNameEx"
   Assembly="BPA.ConfigCollector.dll"
   Class="Microsoft.WindowsServerSystem.BestPracticesAnalyzer.Extensions.GetUserNameExObjectProcessor"
/>

An example object:

<Object Type="GetUserNameEx">
   <Setting Key1="NameFullyQualifiedDN" />
   <Setting Key1="NameSamCompatible" Substitution="CURRENTLOGON" />
   <Setting Key1="NameDisplay" />
   <Setting Key1="NameDnsDomain" />

Specific Object Attributes (see Object for a full attribute list)

Attribute Description
Type This must be set to "GetUserNameEx".

Specific Setting Attributes (see Setting for a full attribute list)

Attribute Description
Key1 This can be one of the following (all of which are fairly self-explanatory):
  • "NameCanonical"
  • "NameCanonicalEx"
  • "NameDisplay"
  • "NameDsnDomain"
  • "NameFullyQualifiedDN" 
  • "NameSamCompatible"
  • "NameServicePrinciple"
  • "NameUniqueId"
  • "NameUserPrinciple"
