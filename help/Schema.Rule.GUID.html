Best Practices Analyzer Core Library
The GUID Attribute

This is a GUID associated with a web or help file article that contains more information on the issue generated.

Example Configuration Fragment

<Rule
   Name="fExPhysicalMachineUp"
   Query="$fClusterServiceRunning and not($szPSWinVer)"
   Error="Error"
   Sev="3"
   Title="Exchange cluster node could not be contacted"
   Text="Exchange cluster node {2} is down or unreachable. This could be due to a network or permissions problem."
   P2="$szExchangeNodeName"
   GUID="fa727882-9fdd-4664-9d49-f0ef95193b49"
   AlwaysEvaluate="True"
/>

Parent Element

Element Description
Rule This element contains information about a rule to execute on the output data.
Template This element identifies a rule template that can be applied to other rules so similar rules don't need to be defined separately.

Possible Values

Element Description
<guid> This identifies a link to detailed article for this issue.
<not present> This means no detailed article exists.
