Best Practices Analyzer Core Library
The ExtFunction Element

This element identifies a function to be loaded at run time and associates that function with a name.  These are used to define custom functions for use in rule queries.

Several functions are provided by default.  See the Function Processors section  for a complete description of these.

Example Configuration Fragment

In the configuration file:

<ExtFunction
   Name="list-average"
   Assembly="BPA.ConfigCollector.dll"
   Class="Microsoft.WindowsServerSystem.BestPracticesAnalyzer.Extensions.PdhObjectProcessor"
   Function="Average"
   Returns="String"
   Parameters="String"
/>
 

Parent Elements

Element Description
Configuration ExtFunction elements only exist in the input configuration.  They are not echoed to the output data file.

Child Elements

Element Description
none This element has no children.

Attributes

Attribute Description
Name This is the name used in a query to invoke the function.
Assembly This should point to a DLL that contains a managed assembly which implements the function.
Class This is the name of the managed class that implements the function.
Function This is the name of the managed function that implements the function.
Returns This identifies what the function returns.
Parameters This identifies what parameters the function expects.
