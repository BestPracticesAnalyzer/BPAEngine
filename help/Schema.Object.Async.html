The Async Attribute

This indicates how many threads should be spun up to process the instances of this object.  This value may not exceed the MaxThreads specified for the entire document.  Care should be taken when async objects are nested under other async objects since it could result in a thread starvation problem.

Example Configuration Fragment

<Object
   Type="Directory"
   Key2="cn=Administrative Groups,%DN%"
   Key3="(&amp;(objectCategory=msExchAdminGroup)%ADMINGROUP%)"
   Key4="OneLevel"
   Class="Admin Group"
   Async="%35"
>

Parent Element

Attribute Description
Object Async attributes are specified on input Object elements.

Possible Values

Value Description
0 This means to spin up one thread per instance, up to the specified maximum.  If not all instances can be dispatched at once, they are queued until othes complete.
1 This is the default value.  It indicates the instances should be processed one at a time.
nn% This specifies a maximum number of threads as a percentage of the MaxThreads specified for the whole run.
n This specifies a fixed number of threads to spin up at most.  If this is great than the specified maximum, it will default to the specified maximum.
