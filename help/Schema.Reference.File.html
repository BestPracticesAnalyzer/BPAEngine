The File Attribute

This identifies a BPA configuration file to retrieve the Type from.  If this is not the current file, then this name must also be included in the ReferencedFiles attribute so updates can be detected.

Example Configuration Fragment

<Reference 
   Name="NOTEPAD.exe File"
   Type="File"
   File="BPA.CommonTypes.xml"
   CustomId="NOTEPAD"
>

Parent Element

Element Description
Reference This element refers to a Type element to be included in that position.

Possible Values

Element Description
<BPA config file name> This is the BPA configuration file to access the Type from.
