Best Practices Analyzer Core Library
The LowerCase Format Processor

This format processor will convert all the text of a value into lower case.

Example Configuration Fragment

The FormatProcessor definition:

<ExtFormat
   Name="LowerCase"
   Assembly="BPA.Common.dll"
   Class="Microsoft.WindowsServerSystem.BestPracticesAnalyzer.Common.CacheObjectProcessor"
   Convert="ToLower"
/>

An example usage:

<Setting
   Key1="NodeName"
   Substitution="NODENAME"
   SubstitutionFormat="LowerCase"
/>

Specific ExtFormat Attributes (see ExtFormat for a full attribute list)

Convert

This should be set to "ToLower".
