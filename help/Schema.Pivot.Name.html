The Name Attribute

Example Configuration Fragment

<Pivot
   Name="ControlServer"
   Key="../@Name"
   Remote="Server"
   KeyRef="$szBaselineControl[../../Value=$szExchangeServerNameCaps]"
/>

Parent Element

Element Description
Pivot This element allows a indirect referencing between rules.

Possible Values

Value Description
<string>This is used by other pivots and rules to references the pivot. It must be unique across all rules and pivots. It must begin with a letter, and contain only letters and digits.
