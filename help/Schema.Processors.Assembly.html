Best Practices Analyzer Core Library
The Assembly Attribute

This attribute is defined on processor elements and identifies the assembly that contains the class to be used.

Example Configuration Fragment

<ConfigPreprocessor
   ObjectType="1 Baseline Preprocessor"
   Assembly="ExBPA.Shared.dll"
   Class="Microsoft.WindowsServerSystem.BestPracticesAnalyzer.Exchange.BaselineConfigPreprocessor"
/>

Parent Element

Attribute Description
ConfigPreprocessor This element identifies a managed class that will be used to preprocess the configuration.
DataPostprocessor This element identifies a managed class that will be used to postprocess the data document after collection and analysis has been completed.
ExtFormat This element maps an Setting.Format to a managed class.
ExtFunction This element maps a Rule.Function to a managed class.
IssueProcessor This element identifies a managed class that will be invoked when a rule fires.
ObjectProcessor This element maps an Object.Type to a managed class.

Possible Values

Value Description
<DLL name> This can be any executable that contains a processor to be loaded into the engine.
