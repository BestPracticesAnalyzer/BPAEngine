The Name Attribute

This is a descriptive name of the object.

Example Configuration Fragment

<Object
   Type="Resolve"
   Name="Server Name"
   Key1="%SRVNAME%,%EXSRVNAME%"
>

Parent Element

Element Description
Instance This attribute will be generated on output Instance elements.
Object This attribute is specified on input Object elements.

Possible Values

Element Description
<property name> This indicates a property to be retrieved for each instance and the value of the property will be the value set on the Name attribute of the instance. Only certain object types support this feature.
<any string> This is the text to set on the Name attribute for all instances created.
<not present> For object types that support this feature, this will typically default to a well known property name that exists on most objects of that kind and is unique.
