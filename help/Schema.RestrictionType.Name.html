Best Practices Analyzer Core Library
The Name Attribute

This is a unique name to apply to this restriction.

Example Configuration Fragment

<RestrictionType
   Name="Task"
   Display="Task"
   Description=""
   Collapse="False"
>

Parent Element

Element Description
RestrictionType This element identifies a restriction to apply to the configuration.

Possible Values

Value Description
<string> This should start with a letter and consist of letters, digits, and dashes.
