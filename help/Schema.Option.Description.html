Best Practices Analyzer Core Library
The Description Attribute

This is an informational description of the option.

Example Configuration Fragment

<Option
   Name="Health"
   Display="Health Check"
   Description="Perform checks for common problems and misconfiguration."
   Cost="90"
   Includes="General,3"
/>

Parent Element

Element Description
Option This element identifies an option of a restriction.

Possible Values

Value Description
<any string> This can be any displayable string.
