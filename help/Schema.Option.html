Best Practices Analyzer Core Library
The Option Element

This element identifies a single option of a given restriction.

Example Configuration Fragment

In the configuration file:

<Option
   Name="Health"
   Display="Health Check"
   Description="Perform checks for common problems and misconfiguration."
   Cost="90"
   Includes="General,3"
/>

Parent Elements

Element Description
RestrictionType This element identifies a restriction to apply to the configuration.

Child Elements

Element Description
none This element has no children.

Attributes

Attribute Description
Cost This provides a hint to the application how long each object that matches the restriction is expected to take.  This can be used to come up with rough estimates of run time.
Description This is an informational description of the purpose of this option.
Display This is the value that should be displayed through the UI to allow a user to choose this option or not.
Group This identifies a parent restriction option.
Includes This is a comma-separated list of options of other restrictions that should be included when this option is enabled. This effect is transitive.
Name This is the name of the option. It must be unique among all options specified in the configuration.
