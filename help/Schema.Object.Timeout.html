The Timeout Attribute

This attribute indicates the timeout to be used for this object.  If processing does not complete in this time, it will be aborted.  If this is not given, the default timeout specified for the run will be used. 

Example Configuration Fragment

<Object
   Type="Registry"
   Key1="%SRVNAME%"
   Key3="Cluster\Resources"
   Key4=".*"
   Key5="Onelevel"
   Timeout="%300"
>

Parent Element

Element Description
Object These attributes are specified on input Object elements.

Possible Values

Value Description
nn% This specifies a maximum timeout, in seconds, as a percentage of the DefaultTimeout specified for the whole run.
n This specifies a fixed timeout, in seconds.
<not present> The default value is the DefaultTimeout specified for the run.
