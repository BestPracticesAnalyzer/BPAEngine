Best Practices Analyzer Core Library
The Type Attribute

This identifies the name of the Type that is being referenced.

Example Configuration Fragment

<Reference
   Type="File"
   File="BPA.CommonModels.xml"
   Name="NOTEPAD.EXE file"
>

Parent Element

Element Description
Reference This element refers to a Type element to be included in that position.

Possible Values

Element Description
<any existing type name> This should match an existing Type Name in the file specified by the File attribute, or in the current file if that attribute is not given.
