The Substitution Element

This element identifies a Substitution variable to be referenced globally.  This is usually used to enable command line parameters for handling configuration preprocessing, but it could be applied in other ways as well.  Substitutions are also used for remapping substitution names from a Reference element to a Type element.  See the descriptions of those elements for details.

For global Substitution element, the inner text of this element contains the value to assign to the substitution by default.  For Reference substitutions, these define the name to assign to all substitution references in the refered to Type.  In this case, the name can either be another substitution (including the percent signs) or it can be a literal value. 

Example Configuration Fragment

In the confgiuration file:

<Substitution
   Name="SERVER">
     (cn=*)
</Substitution>

Parent Elements

Element Description
Configuration This element contains some general meta-information about the input configuration or the output data.
Reference This element refers to a Type element to be included in that position.

Child Elements

Element Description

Attributes

Attribute Description
