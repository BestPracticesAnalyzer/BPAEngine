Best Practices Analyzer Core Library
The IssueProcessor Element

This identifies an issue processor module to be loaded at run time.  Each issue processor configured will be invoked whenever a rule is fired by the analysis engine.. 

See Issue Processors for a complete description of the issue processors that are delivered with the core library.

Child Elements

Element Description
none This element has no children.

Attributes

Attribute Description
Name This is the name used to identify the issue processor.
Assembly This should point to a DLL that contains a C# assembly which has one or more issue processors defined in it.
Class This is the name of a class that has implemented the IssueProcessor logic.
