Best Practices Analyzer Core Library
The Pass Attribute

This only occurs in the output file, and only on rules which were evaluated. 

Example Configuration Fragment

<Rule
   Name="iStatSummaryTotalADSites"
   Query="sum($iStatADSite)"
   Error="None"
   Text="Found {9} Active Directory Site(s)."
   P9="sum($iStatADSite)"
   AlwaysEvaluate="True"
   Pass="False"
>

Parent Element

Element Description
Rule This element contains information about a rule to execute on the output data.

Possible Values

Element Description
"True" This means the Query evaluated to true and the rule generated a message.
"False" This means the rule didn't evaluate to true.
"Unknown" This means the rule doesn't have a Text attribute.
