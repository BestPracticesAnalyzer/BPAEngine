Best Practices Analyzer Core Library
The Group Object Processor

This object can be used to organize the XML document.  It is basically a no-op that will group other objects into logical sets and hierarchies.

This will create exactly one Instance element. The Name attribute for the instance is taken from the object.  Setting elements are not required but if present will be copied to the data document with an empty value.

Example Configuration Fragment

The ObjectProcessor definition:

<ObjectProcessor
   ObjectType="Group"
   Assembly="BPA.Common.dll"
   Class="Microsoft.WindowsServerSystem.BestPracticesAnalyzer.Common.GroupObjectProcessor"
/>

An example object definition:

<Object
   Type="Group"
   Name="AD Sites Connectivity"
   LogText="Processing Active Directory Sites..."
   SecurityContext="ADLOGON"
>

Specific Object Attributes (see Object for a full attribute list)

Attribute Description
Name This is just used for informational purposes.
Type This must be set to "Group".

Specific Setting Attributes (see Setting for a full attribute list)

Attribute Description
None Setting attributes are ignored on Group objects.
