The Name Attribute

This is a unique name to apply to this option.  Note that this should be unique across all restrictions.  This name also represents the name of the attribute to use when applying this restriction.  Restriction attributes can be specified on Objects, Settings, Rules, Templates, and Pivots.

Example Configuration Fragment

<Option
   Name="Health"
   Display="Health Check"
   Description="Perform checks for common problems and misconfiguration."
   Cost="90"
   Includes="General,3"
/>

Parent Element

Element Description
Option This element identifies an option of a restriction.

Possible Values

Value Description
<string> This should start with a letter and consist of letters, digits, and dashes.
