Best Practices Analyzer Core Library
The Type Attribute

This indicates which object processor should be dispatched to process this object.  See Object Processors for a list of all the object processors that are delivered as part of the core library.

Example Configuration Fragment

<Object
   Type="Registry"
   Key1=""
   Key2="HKEY_CURRENT_USER"
   Key3="Software\Microsoft\Exchange\ExBPA\Baseline"
   Key4=".*"
>

Parent Element

Element Description
Object These attributes are specified on input Object elements.

Possible Values

Element Description
<any defined type> This can be any type that has been mapped to an ObjectProcessor.
