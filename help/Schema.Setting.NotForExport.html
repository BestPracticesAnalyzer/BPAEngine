The NotForExport Attribute

If the export operation is selected, then it will strip off all values for this setting (all characters will be replaced with an asterisk).  This allows customers to send scrub their data files of sensitive information before sharing it externally.

Example Configuration Fragment

<Setting
   Key1="ntSecurityDescriptor"
   Format="SecurityDescriptor"
   NotForExport="True"
 />

Parent Element

Element Description
Setting This element identifies a property to be retrieved from an Object.

Possible Values

Element Description
"True" This property is not to be exported.
<anything else> This property can be exported.
