Best Practices Analyzer Core Library
The LogText Attribute

This provides text that the engine will log when it processes this object.  It can be used to better track the progress of a scan while it is running.

Example Configuration Fragment

<Object
   Type="Registry"
   Key1="%SRVNAME%"
   Key3="Software\Microsoft\Windows NT\CurrentVersion"
   LogText="Processing === Exchange Server: %SRVNAME% ==="
   TrackProgress="TrackFailHard"
>

Parent Element

Element Description
Object This attribute is specified on input Object elements.

Possible Values

Value Description
<any printable string> This is the string for the engine to log when processing an object.  Note that you can specify Substitutions in this string.
