Best Practices Analyzer Core Library
The list-percentile Function Processor

This function processor can be used to get the value at or above the specified percentile of the integer nodes in a comma-separated list.  This was designed specifically for use with the Pdh object processor, but it can be applied to any list of integers.

Example Configuration Fragment

The FunctionProcessor definition:

<ExtFunction
   Name="list-percentile"
   Assembly="BPA.ConfigCollector.dll"
   Class="Microsoft.WindowsServerSystem.BestPracticesAnalyzer.Extensions.PdhObjectProcessor"
   Function="ListPercentile"
   Returns="String"
   Parameters="String,Number"
/>

An example usage:

<Setting Key1="SMTP Server" Key2="Cat: Messages Categorized" Key3="_Total" >
   <Rule Name="fMessCatBottom10Percent" Query="list-percentile($.,10))" />
</Setting>

Specific ExtFunction Attributes (see ExtFunction for a full attribute list)

Function

This must be set to "ListPercentile".
Returns This must be set to "String".  This returns the value at or above of the specified percentage of values of the list given.
Parameters This must be set to "String,Number".  This takes a comma-separated list of integers.
