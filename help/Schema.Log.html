Best Practices Analyzer Core Library
The Log Element

This element contains an informational or error message generated during a run.  This data is also echoed to a separate text file.  There will be one of these generated for every message logged.

The inner text of this element contains the actual message logged.

Example Configuration Fragment

In an output data file:

<Log Time="23:20:51.574">
     Processing Active Directory Connector Connection Agreements...
</Log>

Parent Elements

Element Description
Run This output element contains general information about a particular run.

Child Elements

Element Description
none This element has no children.

Attributes

Attribute Description
Time The time the message was logged.
