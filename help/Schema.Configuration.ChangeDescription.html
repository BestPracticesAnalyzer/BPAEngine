The ChangeDescription Attribute

This attribute can be specified on the Configuration element and it allows a developer to present information to a user on what changes are available if they choose to download the new version of the configuration file.  If it can be assumed that people will remain relatively up to date this can be description of only the changes made in the current release rather than all changes made ever. 

The customizable GUI will present this on the screen when prompting the user to decide if they want to download it or not.
