Best Practices Analyzer Core Library
The Key Attribute

For both Object and Setting elements, you can set Key1 to Key5 to indicate exactly what the object processor should do.  The exact meanings and possible values of these attributes is dependent on the object processor. 

Example Configuration Fragment

<Object
   Type="WMI"
   Name="__RELPATH"
   Key1=""
   Key2="root\directory\ldap"
   Key3="rootDSE"
   Key5="*"
>

Parent Element

Element Description
Instance These attributes are copied to the output with the substitutions filled in.
Object These attributes are specified on input Object elements.
Setting These attributes are specified on input Setting elements and copied to the output with substitutions fiilled in.

Possible Values

Value Description
<any> See Object Processors for a list of object processors delivered with the BPA library and how they interpret these keys.
