Best Practices Analyzer Core Library
Common Extensions

The BPA core engine provides a number of extensibility points that enables product teams to add product-specific functionality to a scan.  Each extensibility point is defined using a configuration element.  These are the currently supplied extensibility points:

Config Preprocessors This allows for modifying the input configuration prior to execution.
Data Postprocessors This allows for special processing of the data file after it has been collected and analyzed.
Format Processors This allows for special formatting of retrieved data.
Function Processors This allows for custom functions for use in rules.
Issue Processors This allows for special processing when a rule is fired by the analysis engine.
Object Processors This allows data to be gathered from data sources during the data gathering phase.
