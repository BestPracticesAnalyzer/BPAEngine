Best Practices Analyzer Core Library
The PdhCollect Object Processor

This is used to collect performance data to a blg file for later analysis.  The file can be analyzed with the Pdh object processor.

This object does not create any instances.

Example Configuration Fragment

The ObjectProcessor definition:

<ObjectProcessor
   ObjectType="PdhCollect"
   Assembly="BPA.ConfigCollector.dll"
   Class="Microsoft.WindowsServerSystem.BestPracticesAnalyzer.Extensions.PdhCollectObjectProcessor"
/>

An example object:

<Object
   Type="PdhCollect"
   Key1="%SRVNAME%"
   Key2="10"
   Key3="10"
   Key4="datafile.blg"
/>
 

Specific Object Attributes (see Object for a full attribute list)

Attribute Description
Key1 The name of the server to access.  If processing an already created blg file, this can also be set to "*", in which case it will only process the first server found in the file.
Key2 The number of samples to take.
Key3 This is the period between samples (in seconds). 
Key4 Name of the file to create. 
Type This must be set to "PdhCollect".

Specific Setting Attributes (see Setting for a full attribute list)

    No settings are created as a result of this object execution.
