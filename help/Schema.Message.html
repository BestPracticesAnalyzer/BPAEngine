Best Practices Analyzer Core Library
The Message Element

This element identifies a message generated as the result of a rule evaluating to true.  It is only created on output.

The inner text of this element is the message generated.

Example Configuration Fragment

In an output data file:

<Message
   Name="iStatSummaryTotalADDomains"
   Error="None"
   Sev=""
   GUID="">
      Found 3 Active Directory Domain(s).
</Message>

Parent Elements

Element Description
Setting This element contains data about a particular property to be retrieved.

Child Elements

Element Description
none This element has no children.

Attributes

Attribute Description
Error This value is inherited from the Error attribute of the rule that created it.
GUID This value is inherited from the GUID attribute of the rule that created it.
Name This value is inherited from the Name attribute of the rule that created it.
Sev This value is inherited from the Sev attribute of the rule that created it.
Title This value is inherited from the Title attribute of the rule that created it.
