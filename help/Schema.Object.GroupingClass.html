The GroupingClass Attribute

This provides a display hint to the GUI for showing the object.  It indicates whether the instances under this object should be used to group issues found underneath them.  If true, there will be one group per instance of the object, and each grouping will be labeled using the Class attribute value plus the instance Name value.

Example Configuration Fragment

<Object
   Type="Directory"
   Key2="%DN%"
   Key3="(&amp;(objectCategory=msExchExchangeServer)%SERVER%)"
   Class="Server"
   GroupingClass="True"
   Async="%65"
   TrackProgress="TrackProgress"
   MaxTime="ALL,1"
   MaxFailures="WMI,1"
   MaxTimeouts="ALL,1"
>

Parent Element

Element Description
Object This attribute is specified on input Object elements.

Possible Values

Element Description
"True" This object should be used for grouping.
<any> or <not present> Anything else means do not use this for grouping.  This is the default.
