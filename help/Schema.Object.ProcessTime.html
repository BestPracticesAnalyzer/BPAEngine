The ProcessTime Attribute

This is generated on output and indicates how long it took to process the given object.  Note that this time only counts the time to process the object directly.  It is not a cumulative total of the time to process all objects below it.

Example Configuration Fragment

 <Object
   Type="WMI"
   Key1=""
   Key3="Win32_ComputerSystem"
   ProcessTime="230.8648"
>

Parent Element

Element Description
Object This attribute is generated on output Object elements.

Possible Values

Element Description
<time> This is the time, in milliseconds, that the object took to process.
