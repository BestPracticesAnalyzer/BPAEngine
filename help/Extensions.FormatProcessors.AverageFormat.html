Best Practices Analyzer Core Library
The AverageFormat Format Processor

This format processor will return the average of all numeric values for a Setting.  It is designed to work in conjunction with the Pdh object processor to reduce the amount of data that needs to be stored in the XML.

Example Configuration Fragment

The FormatProcessor definition:

<ExtFormat
   Name="Average"
   Assembly="BPA.ConfigCollector.dll"
   Class="Microsoft.WindowsServerSystem.BestPracticesAnalyzer.Extensions.PdhObjectProcessor"
   Convert="AverageFormat"
/>

An example usage:

<Setting
   Key1="PerfCounterA"
   Substitution="AverageA"
   SubstitutionFormat="Average"
/>

Specific ExtFormat Attributes (see ExtFormat for a full attribute list)

Convert

This should be set to "AverageFormat".
