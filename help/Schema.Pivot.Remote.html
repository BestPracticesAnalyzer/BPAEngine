Example Configuration Fragment

<Pivot
   Name="ControlServer"
   Key="../@Name"
   Remote="Server"
   KeyRef="$szBaselineControl[../../Value=$szExchangeServerNameCaps]"
/>

Parent Element

Element Description
Pivot This element allows a indirect referencing between rules.

Possible Values

Value Description
<string> This is used by indirect pivots to specify the name of the remote pivot. Any references using an indirect pivot will actually refer to a rule under the pivot whose Name matches this value, and whose KeyValue matches this pivot's KeyRefValue.
