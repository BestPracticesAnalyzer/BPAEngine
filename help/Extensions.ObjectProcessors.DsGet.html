Best Practices Analyzer Core Library
The DsGet Object Processor

This retrieves the Active Directory site name for the current or specific server.

Example Configuration Fragment

The ObjectProcessor definition:

<ObjectProcessor
   Name="DsGet"
   Assembly="BPA.ConfigCollector.dll"
   Class="Microsoft.WindowsServerSystem.BestPracticesAnalyzer.Extensions.DsGetObjectProcessor"
/>

An example object:

<Object
   Type="DsGet"
   Key1="SiteName"
   Key2=""
>
   <Setting Key1="SiteName" Substitution="SITENAME" />

Specific Object Attributes (see Object for a full attribute list)

Attribute Description
Key1 Currently, the only supported value for this is "SiteName".
Key2 This is the name of the server to get the site name for.  If not given, this will use the current server.
Type This must be set to "DsGet".

Specific Setting Attributes (see Setting for a full attribute list)

Attribute Description
Key1 Currently, the only supported value for this is "SiteName".  The value of this setting will be the name of the site the AD server is in.
