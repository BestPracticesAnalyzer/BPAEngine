Best Practices Analyzer Core Library
The Resolve Object Processor

This object type can be used to automatically choose between resolution methods (WINS or DNS) depending on which one succeeds.

It expects a single Setting element which it will provide with the value of the resolved name.  Any other Setting elements will also be copied over and given the same value.

Example Configuration Fragment

The ObjectProcessor definition:

<ObjectProcessor
   ObjectType="Resolve"
   Assembly="BPA.Common.dll"
   Class="Microsoft.WindowsServerSystem.BestPracticesAnalyzer.Common.ResolveObjectProcessor"
/>

An example object:

<Object
   Type="Resolve"
   Name="Server Name"
   Key1="%SRVNAME%,%EXSRVNAME%"
>
     <Setting
         Key1="Dns"
         Substitution="SRVNAME"
         Format="^$----%EXSRVNAME%"
     >

Specific Object Attributes (see Object for a full attribute list)

Attribute Description
Key1 This contains a comma-separated list of names to be resolved.  For instance, you might pass in the short name of a server and the FQDN as well.
Type This must be set to "Resolve".

Specific Setting Attributes (see Setting for a full attribute list)

Attribute Description
None The Key attributes of the setting don't matter for this object.  Any setting will be given the same value, which is the resolved name.
