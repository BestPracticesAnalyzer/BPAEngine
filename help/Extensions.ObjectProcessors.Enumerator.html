The Enumerator Object Processor

This enumerator object processor will simply create a certain number of instances.  A setting value at each instance will represent that number of that particular instance.  This basically mirrors how a FOR loop works.

Example Configuration Fragment

The ObjectProcessor definition:

<ObjectProcessor
   ObjectType="Enumerator"
   Assembly="BPA.Common.dll"
    Class="Microsoft.WindowsServerSystem.BestPracticesAnalyzer.Common.EnumeratorObjectProcessor"
/>

An example object:

<Object
   Type="Enumerator"
   Key1="1"
   Key2="25"
   Key3="1"
>
   <Setting Key1="Number" />

Specific Object Attributes (see Object for a full attribute list)

Attribute Description
Key1 This is the starting value to use.  It must be an integer.
Key2 This is the ending value to use.  It must be an integer greater than or equal to Key1.
Key3 This is the increment to use.  It must be a positive integer.  If the value is below 1, it will be forced to equal 1.
Type This must be set to "Enumerator".

Specific Setting Attributes (see Setting for a full attribute list)

Attribute Description
Key1 This can be any name, but the result will be the same: the value of the number for that instance.
