The Customize Element

This element allows a Reference to override certain constructs within the Type definition it references. 

Note that Type, Reference, and Customize elements are only processed when the Type Configuration Preprocessor is configured.  See the built-in Configuration Preprocessor extensions for details.

Example Configuration Fragment

<Customize
   ID="ReferenceName_szRuleName"
   Type="Override"
>
     <Rule
          EventID="1055"
       >
       ...

Parent Elements

Element Description
Reference This element refers to a Type element to be included in that position.

Child Elements

Element Description
Object This element contains information about a single item to be processed.
Reference This element refers to a Type element to be included in that position.
Rule This element contains information about a rule to execute on the output data.
Setting On input, Setting elements are nested directly under Object elements.  These elements contain data about a particular property to be retrieved.

Attributes

Attribute Description
ID This identifies the element in the Type to be customized.  It refers to the CustomId of that element.
Type This is the type of customization to be done.
