Best Practices Analyzer Core Library
The list-max Function Processor

This function processor can be used to get the average value of the integer nodes in a comma-separated list.  This was designed specifically for use with the Pdh object processor, but it can be applied to any list of integers.

Example Configuration Fragment

The FunctionProcessor definition:

<ExtFunction
   Name="list-max"
   Assembly="BPA.ConfigCollector.dll"
   Class="Microsoft.WindowsServerSystem.BestPracticesAnalyzer.Extensions.PdhObjectProcessor"
   Function="ListMax"
   Returns="Number"
   Parameters="String"
/>

An example usage:

<Setting Key1="SMTP Server" Key2="Cat: Messages Categorized" Key3="_Total" >
   <Rule Name="fMessCatIncrease" Query="not (list-average($.)=list-max($.))" />
</Setting>

Specific ExtFunction Attributes (see ExtFunction for a full attribute list)

Function

This must be set to "ListMax".
Returns This must be set to "Number".  This returns the maximum value found.
Parameters This must be set to "String".  This takes a comma-separated list of integers.
