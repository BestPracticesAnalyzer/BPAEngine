Best Practices Analyzer Core Library
The Display Attribute

This indicates the value to display to the user when giving them the option of applying this restriction.

Example Configuration Fragment

<RestrictionType
   Name="Task"
   Display="Task"
   Description=""
   Collapse="False"
>

Parent Element

Element Description
RestrictionType This element identifies a restriction to apply to the configuration.

Possible Values

Value Description
<any string> This can be any displayable string.
