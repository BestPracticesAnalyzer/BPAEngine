The Class Attribute

This provides a display hint to the GUI for showing the object.  It indicates what type of object it is.  The customizable GUI uses this, along with the GroupingClass attribute and the Name attributes on each Instance element to determine how to group issues in a report and what the group headers should look like.

Example Configuration Fragment

<Object
   Type="Directory"
   Key2="%DN%"
   Key3="(&amp;(objectCategory=msExchExchangeServer)%SERVER%)" 
   Class="Server"
   GroupingClass="True"
   Async="%65"
   TrackProgress="TrackProgress"
   MaxTime="ALL,1"
   MaxFailures="WMI,1"
   MaxTimeouts="ALL,1"
>

Parent Element

Element Description
Object This attribute is specified on input Object elements.

Possible Values

Value Description
<any string> Any displayable string (such as "Server"),
