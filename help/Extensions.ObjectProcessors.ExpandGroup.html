Best Practices Analyzer Core Library
The ExpandGroup Object Processor

This returns all the members of an Active Directory group.

Example Configuration Fragment

The ObjectProcessor definition:

<ObjectProcessor
   ObjectType="ExpandGroup"
   Assembly="BPA.ConfigCollector.dll"
   Class="Microsoft.WindowsServerSystem.BestPracticesAnalyzer.Extensions.ExpandGroupObjectProcessor"
/>

Specific Object Attributes (see Object for a full attribute list)

Attribute Description
Key1 This is the prefix to apply to the given DN.  For root level entries, this will default to LDAP://<gc>/rootDSE, and LDAP:// for other entries.  The <gc> value comes from a global substitution.
Key2 This is the DN of the group to be expanded.  It will get appended to Key1.
Key3 This can be one of the following:
  • "Leaf" means that only non-group members should be included in the output.  This is the default.
  • "Inner" means that only group members should be included in the output.
  • "Both" means that all members should be included in the output.
Type This must be set to "ExpandGroup".

Specific Setting Attributes (see Setting for a full attribute list)

Attribute Description
Key1 This is the name of the property to retrieve.
