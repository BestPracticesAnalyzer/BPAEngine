The Display Attribute

This attribute is used as a display hint for an object or setting.  This attribute can have any meaning that is agreed upon between the configuration and the user interface.  The BPA engine does not use this. 

Example Configuration Fragment

<Object
   Type="Directory"
   Key2="%DN%"
   Key3="objectClass=msExchOrganizationContainer"
   Key4="OneLevel"
   Display="Left"
   LogText="Processing Exchange Organization..."
>

Parent Element

Element Description
Object This attribute is specified on input Object elements and copied to output object elements.
Instance This attribute is also copied to the Instance elements under the Object in the output data file.
Setting This element identifies a property to be retrieved from an Object.

Possible Values

Value Description
"Hide" When specified on an Object, the customizable GUI interprets this to mean the object is not to be shown (in the detailed view).  All child entries will be displayed, however.  When specified on a Setting, it will cause that specific setting to be hidden only.
"Left" When specified on an Object, the customizable GUI interprets this to mean that the object should be displayed as a main entry (in an earlier version of the GUI, the display was split into left and right panes which is where this value came from).  This value has no meaning when used on a Setting.
"Right" When specified on an Object, the customizable GUI interprets this to mean that the object should be displayed as a secondary entry (again, this value is from the old GUI).  This value has no meaning when used on a Setting.
<any> If not using the customizable GUI, this can have any value desired.
