The ID Attribute

This identifies the element to be customized within the referenced Type.

Example Configuration Fragment

<Customize
   ID="szRuleName_ReferenceName"
   Type="Override"
>

Parent Element

Element Description
Customize This element can be used to alter specific portions of referenced types.

Possible Values

Element Description
<a CustomId> This should match an existing CustomId that identifies an element within the Type being referenced.  Note that the CustomId will have the reference name prepended to it, so this should be included in this value.
