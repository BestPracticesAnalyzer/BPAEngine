The Template Attribute

This is the name of a Template used by the rule. The template's attributes are copied to the rule, unless already specified on the rule.

Example Configuration Fragment

<Rule
   Name="fSchemaRecentChange"
   Template="fRecentlyChanged"
   Title="Schema recently changed"
   Text="The Exchange schema was changed {8}."
/>

Parent Element

Element Description
Rule This element contains information about a rule to execute on the output data.
Template This element identifies a rule template that can be applied to other rules so similar rules don't need to be defined separately.

Possible Values

Element Description
<any string> This is for informational purposes only.
