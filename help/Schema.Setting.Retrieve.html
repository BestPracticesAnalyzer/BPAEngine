Best Practices Analyzer Core Library
The Retrieve Attribute

This identifies a special retrieval option to apply to this setting.  These may be processed by the engine or the object processor.

Example Configuration Fragment

<Setting
   Key1="CurrentVersion"
   Retrieve="FailHard"
>

Parent Element

Element Description
Setting This element identifies a property to be retrieved from an Object.

Possible Values

Element Description
"CountOnly" Instead of returning all the actual values of this setting, just return the number of values that exist.  This option is handled by the core engine.
"FailHard" If an error occurs retrieving a property, abort processing the entire tree from the parent object on down.  This option is handled by the core engine.
<any> Any other values may be object processor-specific.
