Best Practices Analyzer Core Library
The Value Element

This element identifies a single value of a setting or rule that has been computed in an output data file.

The inner text of this element is the value retrieved.

Example Configuration Fragment

In an output data file:

<Value>
      Respond = Remote socket is not available
</Value>

Parent Elements

Element Description
Result This output element contains the results of a rule that got processed.
Setting This element contains data about a particular property to be retrieved.

Child Elements

Element Description
none This element has no children.

Attributes

Attribute Description
none This element has no attributes.
