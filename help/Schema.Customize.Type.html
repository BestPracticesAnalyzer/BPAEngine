The Type Attribute

This identifies the type of customization to be applied to the identified element.

Example Configuration Fragment

<Customize
   ID="szRuleName_ReferenceName"
   Type="Override"
>

Parent Element

Element Description
Customize This element can be used to alter specific portions of referenced types.

Possible Values

Element Description
"After" This will cause all the child elements to be inserted after the specified element.
"Before" This will cause all the child elements to be inserted after the specified element.
"Override" This will cause the attributes on the first child element to be written over the attributes on the specified element.  All other child elements will be ignored.
"Replace" This will replace the specified element with the child elements.
"Under" This will place the child elements under the specified element.
