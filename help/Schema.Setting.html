The Setting Element

This element identifies a property or other concept such as a command that needs to be processed for the parent Object.  The exact meaning is up to the object processor that handles it.

Example Configuration Fragment

In the configuration file:

<Object
   Type="Directory"
   Key2="%DN%"
   Key3="(&amp;(objectCategory=msExchExchangeServer)%SERVER%)"
   Async="%65"
   Class="Server"
   GroupingClass="True"
   TrackProgress="TrackProgress"
   MaxTime="ALL,120;WMI,60"
   MaxFailures="Metabase,2"
   MaxTimeouts="WMI,5"
>
     <Setting
         Key1="cn"
         Substitution="EXSRVNAME"
      >

In an output data file:

Same as input.

Parent Elements

Element Description
Instance On output, Settings are nested under the specific instance associated with them.
Object On input, Settings are nested under the parent object.
Reference This element refers to a Type element to be included in that position.

Child Elements

Element Description
Message This output element is generated when a rule fires.
Rule This contains information about a rule to execute on the output data.
Value This output element contains the string contents of a single value.

Attributes

Attribute Description
CustomId This is an identifier that can be referenced by a Customize element to override its properties.  This is only meaningful if the Setting is nested under a Type.  This defaults to the Key1 attribute of the Setting.
Display This property is used as a display hint for the Setting.
Format This indicates which ExtFormat to use to display the data.  See Format Processors for a list of built-in format types.
Key1 - Key5 These values can be given any meaning by the object processor associated with this type of parent object.
NotForExport This indicates the value of this data is sensitive.
Restrict This is an internal attribute only.
Retrieve This performs special functionality when retrieving the data.
<restriction> This is a custom-defined restriction attribute.
SDMDataType This provides a hint to be used during conversion of a BPA Object element to an SDM element.
SDMMultiplicity This provides a hint to be used during conversion of a BPA Object element to an SDM element.
SDMName This provides a hint to be used during conversion of a BPA Object element to an SDM element.
Substitution This provides a simple variable setting mechanism within a configuration.
SubstitutionFormat Like Format, but applies to the value set in the substitution only.
